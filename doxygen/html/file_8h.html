◆ show_room()

int show_room ( room *  rm,
student *  head 
)

Displays the seating arrangement of a room along with student information.

This function prints the room name, exam name, and exam date, followed by the seating arrangement with corresponding student information. It iterates through the rows and columns of the room, matching the coordinates with the students' positions. If a student is found at a particular position, their initials and last name are printed. If a position is empty, "<empty>" is displayed.

Parameters
rmPointer to the room structure.
headPointer to the head of the student list.
