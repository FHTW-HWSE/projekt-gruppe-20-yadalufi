Adds a student to the seating plan and performs seat assignment.

This function is called when the user chooses to add a student to the seating plan. It checks if a room plan has been created first. If a room plan exists, the function allows the user to create a new student and assigns a seat to the student. If all seats are taken or if an error occurs during seat assignment, appropriate messages are displayed.
