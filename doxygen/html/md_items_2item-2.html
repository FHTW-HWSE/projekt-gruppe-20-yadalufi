Description

As a user
I want to be able to generate different occupancy percentages.
So that I can see how many students fit in the room at the specific occupancy percentage.

Acceptance criteria

### 1.

Given that the user has defined the room size and layout
when the user selects a specific occupancy percentage from a dropdown menu
then the system tells the number of students that can fit in the room using that occupancy percentage.

### 2.

Given that the user has generated a room with specific occupancy percentage
when the user wants a different occupancy percentage from the dropdown menu
then the system offers the option to restart creation of the room.
